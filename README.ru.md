# Vellora

Локальный лёгкий просмотрщик и редактор Markdown. **2.0** построен на **Tauri 2 + React + TypeScript + Vite**.

Подробности: [中文 README](README.md) или [English README](README.en.md).

## Кратко

- Открытие `.md` / `.markdown`, чтение, правка исходника, сохранение
- Относительные изображения, локальные Markdown-ссылки, подтверждение HTTP(S)
- Установщик Windows NSIS (WebView2 не вшивается)
- Переход с 1.x (Electron): сначала удалите 1.x, затем установите 2.0

## Разработка

```bash
npm install
npm run dev
npm run dist
```
